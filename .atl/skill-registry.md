```

- Event injection: CGEventPost with kCGTabletEventType / kCGTabletProximityEventType.

- Listen UDP + TCP on the same fixed port (e.g. 4545).
```

### For protocol / wire format tasks

```
- Binary, little-endian, fixed header.
- Version byte MUST be first for forward compatibility.
- Timestamps in nanoseconds (u64) for jitter measurement.
- Same payload format over UDP (Wi-Fi) and TCP (USB via adb reverse tcp:4545).
```
